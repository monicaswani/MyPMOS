# docs/decisions/

Architecture Decision Records (ADRs).

## Conventions

- One decision per file, named `NNNN-short-title.md` (e.g., `0001-use-n8n.md`).
- Capture: context, the decision, status, and consequences.
- ADRs are append-only; supersede rather than rewrite. Mark status as
  `Proposed`, `Accepted`, `Superseded`, or `Deprecated`.
