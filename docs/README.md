# docs/

Project documentation for PMOS.

## Structure

| Subdirectory     | Purpose                                                       |
| ---------------- | ------------------------------------------------------------- |
| `architecture/`  | System architecture and component overviews.                  |
| `decisions/`     | Architecture Decision Records (ADRs).                         |
| `setup/`         | Setup, environment, and contributor guides.                   |
| `api/`           | API and interface documentation.                              |

Start with [`architecture/overview.md`](./architecture/overview.md). Record
notable design decisions as dated ADRs in [`decisions/`](./decisions).
